💡 一句话总结

如果你…… 首选
刚开始搭建工具箱 PowerToys（Win）/ Raycast（Mac）+ Bitwarden + VLC
是前端开发者 VS Code + ImageOptim + draw.io
需要整理笔记 Obsidian
需要录课程/直播 OBS Studio + HandBrake
要剪音频 Audacity
画图设计 GIMP + Inkscape + Krita
日常办公 LibreOffice + Thunderbird + Calibre

原则： 优先开源、取之官网、远离流氓。所有链接均为官方网站，建议从官网下载安装。
