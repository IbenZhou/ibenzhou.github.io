# 0478 Computer Science 2025 May Paper 1 考点预测

> 最近的 Syllabus 是从 2023 年开始执行，回顾近 2 年试卷，有效试卷共15 套，统计所有和定义相关的考点出现频率来预测绷瓷考试中可能出现的考点

1. 必考考点
   
   - Number system conversion
   
   - Binary addition

   - two's complement
   
   - The usage of Hexadecimal
   
   - File size calculation

   - [[Diagram]]

2. 历年出现频率较高的考点
   
   - Transmission method（11 次）- 着重关注几种方法的定义及优缺点分析
   
   - Input & Output devices（10 次）- 掌握常见的设备名称
   
   - Microprocess（9 次）- 该考点一般是和 sensor 结合起来考查，背答题模板
   
   - Packet（9 次）- 着重掌握 packet 结构，packet header 考察频率较高，**定义**
   
   - Compression（8 次）- 考察 Compression 的优缺点分析，重点关注两种 Compression 的定义和优劣势分析，另外重点掌握 RLE
   
   - Expert system（8 次）- 背答题模板
   
   - F-E cycle（7 次）- 该考点的知识点比较多，需要掌握整个流程，中间的 register 的名称和作用都要清楚
   
   - Secondary storage（7 次）- 掌握定义 + 常见设备名称 + 结合 Primary storage 进行优劣势分析
   
   - Parity check（7 次）- 重点掌握定义
   
   - Router（6 次）- 掌握基本定义，重点关注其在不同场景中的角色和作用
   
   - URL（6 次）- 定义+结构
   
   - AI（6 次）- 答题模板
   
   - Cookies（5 次）- 重点掌握其基本结构，对比两种不同类型的 cookies 的不同和具体使用场景
   
   - ARQ（5 次）- 定义
   
   - Operating system（5 次）- OS 的基本管理功能（process，memory，multitasking，file，interrupt...)
   
   - Performance factor（5 次）- 重点掌握集中影响 CPU 性能的常见因素
   
   - Robotics（5 次）- 定义+优缺点分析
   
   - Automated system（5 次）- 定义+优缺点分析
   
   - Encryption（5 次）- 定义，重点关着 symmetric 和 asymmetric 定义原理和异同
   
   - Firewall（5 次）- 定义 + 使用场景
   
   - Programming language（5 次）- 对比 High-level language 和 Low-level language 的优劣势
   
   - IDE（5 次）- 掌握 IDE 的基本功能，以及在 programming 的过程中提供的便利
   
   - Interpreter & Compiler（5 次）- 重点关注定义 + 两者异同对别 + 优劣势分析
   
   - Machine learning（5 次）- 定义
